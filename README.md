# Connect 4

A mobile version of the classic Connect 4 game built using Expo and NativeWind

## Features

- Classic Connect 4 gameplay.
- Responsive and visually appealing UI.
- Cross-platform support (iOS and Android).

## Technologies Used

- **Expo**: For building and running the React Native app.
- **NativeWind**: For Tailwind CSS-inspired styling in React Native.
